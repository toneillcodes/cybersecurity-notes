# Active Directory Playbook
## Overview
- Enumeration using tools
    - LDAP, SMB (NXC)
    - Bloodhound, Plumhound, PingCastle
- AS-REP Roasting
- Kerberoasting
- secretsdump (requires admin)
- Password cracking
- PTH/PTP for lateral movement
- Password spraying
- Repeat
